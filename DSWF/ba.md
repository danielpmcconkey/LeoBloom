# BA Addendum — LeoBloom

Read `DSWF/conventions.md` first — it defines naming conventions and artifact
inventory for all agents.

## Brainstorm Location

`Projects/ProjectNNN-ShortName/ProjectNNN-brainstorm.md`

Use the `ProjectNNN-` prefix matching the directory. Not `PNNN-`.

## When to Brainstorm

Skip if the backlog item is unambiguous and has clear acceptance criteria.
Brainstorm when requirements are ambiguous, have GAAP implications, or
involve domain concepts the team hasn't worked with before.

## GAAP Awareness

LeoBloom is a real accounting system with real money. If the backlog item
touches financial logic and you're unsure about the correct accounting
treatment, flag it as an open question for Dan rather than guessing.
